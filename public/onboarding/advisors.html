Welcome

Welcome to the EBHCS Advisor Bulletin Board.

This is where you post opportunities, events, and resources for students — and where they go to find them.

This guide takes about 5 minutes. Use Next or the arrow keys.

The two big ideas

Bulletins vs. Resources

📣 Bulletins

Your live feed. Timely things: a job opening this month, a workshop next week, a deadline. They show up newest-first.

🧭 Resources

Always there. Permanent help links by topic — housing, immigration, food, health, jobs.

💡 Rule of thumb: if it has a deadline, it's a bulletin. If it'll still be useful next year, it's a resource.

Posting

Post a bulletin

  1. Click Create → Bulletin.
  2. Add a clear title and a short description.
  3. Optional: attach a PDF (max 10MB) or a link.
  4. Hit Publish. It shows up in the student feed instantly.
[ Screenshot: New bulletin form ]
💡 Short titles win. "Summer internship — apply by June 1" beats "Important opportunity for students."
Posting

How the calendar works

There are two ways things end up on the calendar:

📣 Bulletin with a date

If your bulletin has an event date attached (a workshop, a deadline, a fair), it automatically shows up on the calendar too. You only have to post it once.

🗓️ Calendar event

For date-only items that aren't a bulletin — Memorial Day, First Day of School, February Break, early dismissal. No long description needed.

💡 If you'd write a paragraph about it, post a bulletin. If it's just a date students should know, make it a calendar event.
[ Screenshot: Calendar view showing both a bulletin event and a date-only event ]
Posting

Add a resource

Use this for evergreen help — things that stay useful all year. Pick a category (Immigration, Housing, Health, etc.), give it a title, and paste the link.

ℹ️ All advisors can see and reorder resources — try to avoid duplicates. Search the category first.

Know what's working

Analytics

Each of your posts shows views and link clicks. Switch the date range (7 / 30 / 90 days) to see trends.

[ Screenshot: Analytics dashboard with date toggle ]
💡 If a post has lots of views but no clicks, the title is great but the content needs work.
Cleanup

Edit or delete your posts

  • You can edit or delete your own posts any time.
  • Admins can override and edit anyone's — ping Marlie if something needs to come down fast.
