Welcome

Welcome, admin.

This is the EBHCS Advisor Bulletin Board admin guide. You can do everything an advisor can do — post bulletins, calendar events, and resources — plus override anyone's post and manage advisor accounts.

The first half walks the full advisor workflow. The admin-only powers start after that.

The two big ideas

Bulletins vs. Resources

📣 Bulletins

The live feed. Timely things: a Bunker Hill application deadline, a work training next week, a Boston career fair on the 14th. Newest-first.

🧭 Resources

Always there. Permanent help links by topic — Immigration, Housing, Health, Food, ESOL, College & GED, Money Help.

[ Capture as: onboarding-photos/homepage.png — Homepage showing the bulletin feed and Find Help entry point ]
💡 Rule of thumb: if it has a deadline, it's a bulletin. If it'll still be useful next year, it's a resource.

Step 2

The dashboard, in one look

Posting

Post a bulletin

  1. Click Create → Bulletin.
  2. Pick a category (Job, ESOL, Immigration, College & GED, etc.) and write a clear title and short description.
  3. Optional: attach a flyer PDF (max 10MB) or a link.
  4. Hit Publish. It shows up in the student feed instantly — in both English and Spanish.
[ Capture as: onboarding-photos/new-bulletin.png — Create Bulletin form with category dropdown open ]
💡 Short titles win. "ESOL Level 2 — placement test Friday 4pm" beats "Important info about English class."
Posting

How the calendar works

There are two ways things end up on the calendar:

📣 Bulletin with a date

If a bulletin has an event date attached (a workshop, deadline, fair), it automatically shows up on the calendar too. Post it once.

🗓️ Calendar event

For date-only items — Memorial Day, First Day of School, February Break, half-day dismissal, parent–advisor night. No long description needed.

💡 If you'd write a paragraph about it, post a bulletin. If it's just a date students should know, make it a calendar event.
[ Capture as: onboarding-photos/calendar.png — Calendar view with one bulletin-event and one date-only event ]
Posting

Add a resource

Use this for evergreen help — things that stay useful all year. Pick a category (Immigration, Housing, Health, Food, ESOL, College & GED, Money Help, Job Help), give it a title, and paste the link. Example: "DTA SNAP application" under Food, or "Bunker Hill Community College admissions" under College & GED.

ℹ️ All advisors can see and reorder resources — check the category first so we don't end up with two of the same link.

Analytics

Each post shows views and link clicks. Switch the date range (7 / 30 / 90 days) to see trends.

Cleanup

Edit or delete posts

  • Advisors can edit or delete their own posts any time.
  • As admin, you can edit or delete anyone's — use it sparingly and let the original author know (a quick Slack or email to the advisor is plenty).

Admin-only powers
Admin

What admins can do that advisors can't

  • Edit or delete anyone's posts — useful for cleanup, corrections, or take-downs.
  • Manage advisor accounts — add, edit, remove advisors and reset passwords from the Manage Advisors page.
  • Edit office hours & advisor directory on the About page.
ℹ️ Admin accounts today: admin@ebhcs.org and leah@ebhcs.org (Leah Gregory).
Admin

The Manage Advisors page

In the Advisor Portal you'll see a Manage Advisors page (admins only). Everything you need to add, edit, reset, or remove an advisor lives here — no Firebase console required.

[ Capture as: onboarding-photos/manage-advisors.png — Manage Advisors page with Add form and existing advisor cards ]
Admin

Adding a new advisor

  1. Open Manage Advisors in the Advisor Portal.
  2. Fill in the Add New Advisor form:
    • Username — short login name (e.g. jerome, carmen, fabiola).
    • Display name — what students see on bulletins (e.g. Jerome, Mike K.).
    • Email — their @ebhcs.org address.
    • Toggle Admin on only if they should manage all posts and advisors.
  3. Click + Add Advisor.
  4. Share their starting password: ebhcs2025. They'll be asked to change it on first sign-in.
⚠️ Only @ebhcs.org emails work. Anything else will be rejected at login.
Admin

Editing or removing an advisor

Each advisor card on the Manage Advisors page has three buttons:

  • Edit — update their display name, email, or admin status.
  • Reset password — sends them a reset link (or sets it back to the default).
  • Remove — deletes their account. Their existing posts stay; new ones can't be made.
[ Capture as: onboarding-photos/advisor-card.png — Advisor card with Edit / Reset password / Remove ]
💡 The Admin account card has no Remove button — you can't delete the root admin from the UI.
Admin

Removing or overriding a post

When you're logged in as an admin, every post card shows Edit and Delete — even posts you didn't create.
